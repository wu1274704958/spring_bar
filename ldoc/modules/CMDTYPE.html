CMDTYPE

Note, the CMDTYPE[] table is bidirectional.

That means: CMDTYPE[CMDTYPE.ICON] := "CMDTYPE_ICON"

Fields:

  1. ICON number expect 0 parameters in return
  2. ICON_MODE number expect 1 parameter in return (number selected mode)
  3. ICON_MAP number expect 3 parameters in return (mappos)
  4. ICON_AREA number expect 4 parameters in return (mappos+radius)
  5. ICON_UNIT number expect 1 parameters in return (unitid)
  6. ICON_UNIT_OR_MAP number expect 1 parameters in return (unitid) or 3 parameters in return (mappos)
  7. ICON_FRONT number expect 3 or 6 parameters in return (middle and right side of front if a front was defined)
  8. COMBO_BOX number expect 1 parameter in return (number selected option)
  9. ICON_UNIT_OR_AREA number expect 1 parameter in return (unitid) or 4 parameters in return (mappos+radius)
  10. ICON_UNIT_FEATURE_OR_AREA number expect 1 parameter in return (unitid or Game.maxUnits+featureid) or 4 parameters in return (mappos+radius)
  11. ICON_BUILDING number expect 3 parameters in return (mappos)
  12. ICON_UNIT_OR_RECTANGLE number expect 1 parameter in return (unitid) or 3 parameters in return (mappos) or 6 parameters in return (startpos+endpos)
  13. NUMBER number expect 1 parameter in return (number)
  14. CUSTOM number used with CMD_INTERNAL
  15. NEXT number next command page used with CMD_INTERNAL
  16. PREV number previous command page used with CMD_INTERNAL
